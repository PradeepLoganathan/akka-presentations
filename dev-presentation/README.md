# Governance-First AI Dev Presentation

Confidential R&D offsite teaching deck.

## Build

```bash
cd dev-presentation
python builder/build.py
```

Output: `generated/overview/index.html`.

## Narrative Spine

- Governance is foundational for AI because AI behavior is probabilistic.
- Specify consumes both functional requirements and EvalMatrix definitions.
- Specify generates both the functional system and the governance elements.
- Akka converges those generated outputs into one governed AI system.
- The convergence mechanism is intentionally not specified in this deck; it is an R&D concern.
