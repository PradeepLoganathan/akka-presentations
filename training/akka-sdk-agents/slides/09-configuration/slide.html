Agents

All in application.conf.

src/main/resources/application.conf
akka.javasdk {
  agent {
    # Which provider agents default to
    model-provider = "akka.javasdk.agent.openai"

    openai {
      model-name  = "gpt-4o-mini"
      temperature = 0.2
      # api-key = ${?OPENAI_API_KEY}   (inherited default)
    }
  }
}
  • SDK ships default openai/anthropic/gemini configs
  • Override only what you need - model name, temperature, base URL
  • api-key = ${?OPENAI_API_KEY} is already in the reference config
  • Fail-fast on startup if the key is missing
  • Individual agents can override via .model(ModelProvider.openAi(...))
