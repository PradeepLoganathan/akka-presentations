The sample

SupportAgent - the real one.

A customer-support agent that answers "where's my order?" using the sample's data as tools.

Job

Answer questions about a customer's profile, orders, and audit history.

Tools

Four function tools bound to entity + view queries.

Memory

Last 10 turns, keyed by sessionId. Multi-turn context works.

The system prompt is the guardrail: "Never invent ids. Use the provided tools. If a tool returns nothing, say so."
