MCP

MCP: tools your agents didn't write.

Model Context Protocol lets external LLM clients (Claude Desktop, Cursor, other agents) call tools you expose over HTTP.

You expose

Tools, resources, prompts - via annotations on plain Java methods.

Clients call

JSON-RPC over HTTP. Streamable. Discoverable via tools/list.

Auth

Same @Acl annotations as any endpoint. Same identity model.

Two audiences for the same domain: your internal agent uses @FunctionTool; external clients use @McpTool.
