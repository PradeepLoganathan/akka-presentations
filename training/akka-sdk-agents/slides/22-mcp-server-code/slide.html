@McpEndpoint(
    path = "/mcp",
    serverName = "akka-workshop-customer",
    serverVersion = "0.1.0",
    instructions = "Read-only customer and order lookup tools.")
@Acl(allow = @Acl.Matcher(principal = Acl.Principal.INTERNET))
public class CustomerMcpServer {

  @McpTool(description =
      "List all orders for a given customer.")
  public String listOrdersForCustomer(
      @Description("Customer id") String customerId) {
    ...
  }
}
  • @McpEndpoint declares the MCP server at a path
  • serverName / version returned in the initialize handshake
  • @McpTool methods must return String
  • JSON schema is auto-derived from Java params
  • @Description on params gives the LLM context
