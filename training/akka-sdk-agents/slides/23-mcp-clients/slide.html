Wiring a real client - Claude Desktop.

~/.config/claude/claude_desktop_config.json
{
  "mcpServers": {
    "akka-workshop": {
      "transport": "http",
      "url": "http://localhost:9000/mcp"
    }
  }
}
  • Same MCP the Anthropic client already speaks
  • No plugin, no adapter code - just URL + name
  • Restart Claude Desktop; tools appear in the tool picker
  • Same setup pattern for Cursor, other agents, other services
  • Deploy to Akka runtime → publicly reachable MCP endpoint
