Deploy

The akka CLI, in six commands.

Terminal
# auth once per machine
akka auth login

# projects group services + regions + config
akka projects new akka-workshop-prod
akka config set project akka-workshop-prod

# see what regions are available
akka regions list

# see current deployments
akka services list

# tail logs from anywhere
akka logs
  • Single binary, no plugins
  • Auth is persistent per machine
  • Projects scope services, regions, secrets, ACLs
  • Config is namespaced — different terminal, different project
  • Everything the console does is scriptable via CLI
