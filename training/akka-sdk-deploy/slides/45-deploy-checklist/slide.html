Deploy

Before you push the first image.

A short checklist that saves you a Slack apology later.

Secrets

OPENAI_API_KEY, DB creds - via akka secrets, not the image.

Resource limits

Memory + CPU per instance. Sensible defaults; tune based on load tests.

Health checks

SDK provides them; assert readiness before promoting.

ACLs

What's public? Which endpoints require auth? Which are internal-only?

Log level

INFO in prod. DEBUG on demand via config reload.

Rollback path

Know how to akka services deploy the previous tag before you need to.
