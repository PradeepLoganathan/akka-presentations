Multi-region

Entity semantics across regions.

The mental model shifts slightly. Local reads always. Writes go to the entity's home region.

Reads

Always local. Views project per region, entity state is replicated with bounded staleness.

Writes

Routed to the entity's home region — usually one round-trip more if the caller is elsewhere.

Consistency

Single-writer per entity globally. No conflicting writes at the entity level.
