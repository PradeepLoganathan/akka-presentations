Observability

What the runtime exports by default.

Every component emits metrics, structured logs, and traces without any wiring on your part. You can layer custom metrics on top.

Metrics

Prometheus format. Request rate, latency histograms, entity active count, workflow completion.

Logs

Structured JSON. Component id, session id, trace id on every line.

Traces

OpenTelemetry. Endpoint → entity → consumer chain shows up as one trace.

All three are on by default in the managed runtime. In dev, hit /metrics and /actuator/health yourself.
