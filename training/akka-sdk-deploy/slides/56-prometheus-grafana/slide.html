Observability

Prometheus + Grafana: plug it in.

Terminal
# wire the service's metrics to your Grafana Cloud
akka observability metrics export prometheus \
    --url https://prometheus-prod.grafana.net/api/prom/push \
    --username "..." \
    --password-from-secret grafana-key

# or scrape the /metrics endpoint from your own Prometheus
scrape_configs:
  - job_name: akka-workshop
    metrics_path: /metrics
    static_configs:
      - targets: ['workshop-abc.akka.app']
  • Metrics available regardless of Prometheus setup
  • Dashboards: sample project comes with a starter JSON
  • Golden signals: rate, error, duration — per component type
  • Track: workflow queue depth, entity passivation rate, LLM latency
  • Alert on workflow completion time p99 — not just endpoint latency
