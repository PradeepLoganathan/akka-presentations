Observability

Alerts that page a human.

Stateful systems need different alerts. Endpoint 500-rate is one signal; a workflow that hasn't reached terminal state in 24h is another.

SignalWhy it matters
HTTP 5xx rateEndpoints are broken. Classic. Page.
Workflow stuck > N minutesSaga jammed in a step. Data may be inconsistent. Page.
Entity passivation churnCluster instability or misconfigured resource limits. Investigate.
Consumer lag > N secondsEvent backlog. Views + audit going stale. Page.
LLM error rateProvider outage or quota exhausted. Depends: page for prod, ignore in dev.
