Hands-on · 90 minutes

Ship it to your Akka account.

End-to-end deploy. Real image, real service, real URL. By the end of the lab you'll be curling a production endpoint.

Terminal
akka auth login
akka projects new my-workshop-prod
akka config set project my-workshop-prod

# build + push image
mvn install
docker push ghcr.io/<you>/akka-workshop:1.0

# store your OpenAI key
akka secrets create generic openai-key \
    --literal OPENAI_API_KEY=$OPENAI_API_KEY

# deploy
akka services deploy akka-workshop \
    ghcr.io/<you>/akka-workshop:1.0 \
    --secret-env openai-key

# expose + smoke test
akka routes create workshop --service akka-workshop
curl https://workshop-<yours>.akka.app/api/customers
  • Bring your own image registry (GHCR free tier is fine)
  • Bring your OPENAI_API_KEY
  • Bonus: add a second region
  • Bonus: wire /metrics to Grafana Cloud
