Positioning

Not a library. A runtime with an SDK.

You write
  • Components: entities, views, endpoints, workflows, timers, consumers, agents
  • Plain Java records & classes with a few annotations
  • Maven module — nothing exotic
Akka runs
  • Persistence, sharding, cluster membership
  • Message delivery guarantees & back-pressure
  • HTTP surface, ACLs, observability
  • Rolling deploys, autoscale, multi-region
The SDK's job is to keep distribution invisible while you write domain logic.
