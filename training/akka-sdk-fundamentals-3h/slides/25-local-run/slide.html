Ops

Running locally.

Zero-config dev loop. One Maven target, one port, one embedded runtime.

Terminal
# start the service
mvn compile exec:java

# in another shell
curl -XPOST http://localhost:9000/api/customers \
  -H 'Content-Type: application/json' \
  -d '{"customerId":"c-1","FirstName":"Ada","LastName":"Lovelace"}'

curl http://localhost:9000/api/customers
  • No local DB to install
  • No Kubernetes to spin up
  • Console: localhost:9000/akka/console
  • Hot restart on code change: rerun the target
  • The same code will run in the managed Akka runtime
