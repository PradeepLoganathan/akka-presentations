The sample

akka-workshop - one entity, one view, one endpoint.

Everything we cover today maps to a file in this repo. Clone it now - we'll come back to it after every block.

Clone & run
git clone https://github.com/PradeepLoganathan/akka-workshop.git
cd akka-workshop
mvn compile exec:java
  • Local console: http://localhost:9000/akka/console
  • Endpoint: http://localhost:9000/api/customers
  • Java 21 · Akka SDK 3.4
Tree
src/main/java/com/pradeepl/akkakata/
├── domain/
│   ├── commands/   // intent
│   ├── events/     // facts
│   ├── model/      // state records
│   └── entities/   // value + event-sourced
├── views/          // projections
└── api/            // HTTP endpoints
