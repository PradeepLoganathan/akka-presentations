Mental model

Five nouns for the next three hours.

Every component you write is one of these. Keep this vocabulary loaded.

01
Command
A request to change something. Can be rejected.
02
Event
A fact that already happened. Immutable, journaled.
03
Entity
A stateful thing addressed by an ID.
04
View
A read-optimized projection over entity data.
05
Endpoint
The HTTP surface. Talks to entities & views via a client.
+
Later
Workflows, timers, consumers, agents - same shape.
