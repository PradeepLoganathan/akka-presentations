Entities

Value entity: snapshot in place.

Simplest flavor. Handlers replace the whole state. No event log.

domain/entities/CustomerPreferencesEntity.java
@ComponentId("customer-preferences")
public class CustomerPreferencesEntity
    extends KeyValueEntity<CustomerPreferences> {

  @Override
  public CustomerPreferences emptyState() {
    return CustomerPreferences.defaults();
  }

  public Effect<String> set(SetPreferences cmd) {
    var next = new CustomerPreferences(
        cmd.theme(), cmd.locale(), cmd.marketingOptIn());
    return effects()
        .updateState(next)
        .thenReply(__ -> "OK");
  }
}
  • Extends KeyValueEntity<State>
  • No events, no reducer - just updateState(next)
  • Runtime persists the latest state atomically
  • Great for preferences, sessions, feature flags
  • Loses history - last write wins
