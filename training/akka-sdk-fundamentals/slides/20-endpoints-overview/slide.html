Endpoints

HTTP endpoints are the only external door by default.

Entities and views are not directly reachable from the network. Endpoints are the boundary.

@HttpEndpoint

Class-level annotation with a base path.

@Get / @Post / @Put / @Delete

Method-level annotations that map to routes.

@Acl

Access control - who can hit this endpoint.

Endpoints are stateless. All state lives in entities and views. Endpoints just translate HTTP ↔ domain calls.
