Testing

TestKit — unit tests for stateful things.

No cluster, no HTTP. In-JVM, fast, deterministic.

CustomerEntityTest.java
var testKit = EventSourcedTestKit.of(CustomerEntity::new);

var result = testKit.method(CustomerEntity::create)
    .invoke(new CreateCustomer("Ada", "Lovelace"));

assertThat(result.getReply()).isEqualTo("customer-1");
assertThat(result.getNextEventOfType(customerCreated.class))
    .extracting(customerCreated::firstName)
    .isEqualTo("Ada");
  • Instantiates the entity in memory
  • Feeds it commands; asserts on emitted events and reply
  • Chains multiple commands to test evolution
  • No mocking — the real entity, minus the cluster
  • Views & endpoints have their own testkits too
