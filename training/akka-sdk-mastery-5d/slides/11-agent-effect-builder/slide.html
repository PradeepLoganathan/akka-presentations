The Effect.Builder, one call at a time.

SupportAgent.ask()
public Effect<String> ask(String question) {
  return effects()
      .memory(MemoryProvider.limitedWindow().readLast(10))
      .systemMessage(SYSTEM_PROMPT)
      .tools(this)              // scan @FunctionTool methods on this class
      .userMessage(question)
      .thenReply();
}
  • .memory(...) — MemoryProvider (LimitedWindow / Disabled / Custom)
  • .systemMessage(...) — guardrails, role, constraints
  • .tools(this, ...) — objects or classes hosting @FunctionTool methods
  • .userMessage(...) — the actual question
  • .thenReply() — return the model's text response
