SupportAgent's four tools.

Each one hits a component we built on previous days. The LLM chooses which to call.

ToolWhat it queries
getCustomer(id)CustomerEntity.get() — profile summary
listOrdersForCustomer(id)OrdersView.byCustomer — all orders + statuses

The LLM orchestrates tools without your code deciding order. Give clear descriptions and it usually does the right thing.
