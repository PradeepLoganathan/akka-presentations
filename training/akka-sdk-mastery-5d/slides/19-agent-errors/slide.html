Agents

Failures the runtime catches for you.

Agent effects have a first-class error taxonomy. Recover per case, don't blanket-catch.

ExceptionWhen
RateLimitExceptionProvider throttled you. Back off, retry, or degrade.
ModelTimeoutExceptionModel didn't respond in time. Consider a shorter prompt.
ToolCallLimitReachedExceptionModel tried to call tools too many times. Simplify or add a tool.
ToolCallExecutionExceptionYour tool threw. Fix the tool or return a plain error string.
JsonParsingExceptionStructured response didn't parse. Tighten the schema prompt.
InternalServerExceptionProvider 5xx. Retry with backoff.
Use .onFailure(Function<Throwable, String>) to substitute a graceful response per exception type.
