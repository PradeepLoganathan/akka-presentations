Hands-on · 90 minutes

Talk to your service.

The agents branch is fully wired. Set the key, ask questions, connect an MCP client.

Terminal
git checkout agents
export OPENAI_API_KEY=sk-...
mvn compile exec:java

# create fixtures
curl -XPOST http://localhost:9000/api/customers ... -d '{...}'
curl -XPOST http://localhost:9000/api/orders    ... -d '{...}'

# ask the agent
curl -XPOST http://localhost:9000/api/support/s-1 \
  -H 'Content-Type: application/json' \
  -d '{"question":"Where are cust-ada'\''s orders?"}'

# follow up in the same session (memory!)
curl -XPOST http://localhost:9000/api/support/s-1 -d '{"question":"Dig into the FAILED one"}'
