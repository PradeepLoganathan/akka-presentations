WorkflowTestKit — step by step.

OrderWorkflowTest.java (excerpt)
var testKit = WorkflowTestKit.of(OrderWorkflow::new);

var started = testKit.method(OrderWorkflow::start)
    .invoke(new StartOrder("c1",
        List.of(new OrderLine("sku-1", 1, 100))));

assertThat(started.getReply()).startsWith("order-");
assertThat(testKit.currentState().totalCents()).isEqualTo(100L);

// advance one step at a time
var t1 = testKit.getTransition();
assertThat(t1.stepName()).isEqualTo("reserve");

testKit.runStep("reserve");
var t2 = testKit.getTransition();
assertThat(t2.stepName()).isEqualTo("charge");
  • In-JVM, no cluster, no HTTP
  • Mock component clients per step for isolation
  • Assert transitions, state, scheduled timers
  • Run one step at a time or the whole workflow
  • Time is virtual — test 24h timers in milliseconds
