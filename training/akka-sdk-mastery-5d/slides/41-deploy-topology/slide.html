Deploy

What Akka runs for you.

You hand the platform a container image and a service name. The platform provides everything below.

Cluster

Nodes, membership, gossip, sharding. Autoscaled.

Persistence

Event journal, entity snapshots, view tables, workflow state, session memory.

Networking

HTTP ingress, gRPC internal, mTLS between services, TLS termination.

Deployment

Rolling upgrades, health checks, blue/green when needed.

Observability

Metrics, logs, distributed traces, dashboards, alerts.

Multi-region

Replication topology, per-region entities, active/active or active/standby.
