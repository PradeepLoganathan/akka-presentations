Deploy

Building the container image.

Terminal
# builds the jar and packages it as a container image
mvn install

# tag and push to your registry
docker tag akka-workshop:1.0 \
  ghcr.io/pradeepl/akka-workshop:1.0
docker push ghcr.io/pradeepl/akka-workshop:1.0

# confirm the registry has it
docker manifest inspect ghcr.io/pradeepl/akka-workshop:1.0
  • Parent POM adds the JIB plugin — no Dockerfile needed
  • Any OCI registry works: GHCR, ECR, GAR, Docker Hub
  • Registry needs to be reachable by the Akka runtime
  • Private registries: add pull-secret to the project
  • Tag by version, not by latest — deploys should be reproducible
