Deploy

First deploy: akka services deploy.

Terminal
# deploy the service
akka services deploy akka-workshop \
    ghcr.io/pradeepl/akka-workshop:1.0

# wait for it to be ready
akka services list
## STATE: Ready  READY: 3/3

# expose it publicly (routes are project-scoped by default)
akka routes create workshop-route \
    --service akka-workshop \
    --port 9000

# test it
curl https://workshop-abc.akka.app/api/customers
  • Same command; the platform handles rolling deploys after the first
  • Routes give you a public HTTPS URL
  • Custom domains + certs configured per project
  • Health checks and readiness are inferred from the SDK
  • Ready to accept traffic in ~60s from image push
