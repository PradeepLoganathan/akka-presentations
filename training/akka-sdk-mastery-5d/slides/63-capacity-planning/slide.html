Capacity planning: what to size for.

Stateful workloads size on different dimensions than stateless ones. Instance count is only one lever.

Active entities

Entities in memory per node. Passivation reclaims idle ones.

Workflow throughput

How many workflows can transition per second? Set by step call latency, not CPU.

View lag

How stale can reads be? Sets consumer parallelism.

LLM tokens

Per-agent-call. Budget = model choice × memory window × tool calls.

Storage

Journal grows forever unless you archive. Design event schema for retention early.

Egress

LLM calls are egress. Multi-region calls are egress. Both add up.
