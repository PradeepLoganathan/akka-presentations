Agent testing: mock the model.

SupportAgentTest.java (shape)
var tk = AgentTestKit.of(SupportAgent::new)
    .withMockedModel("gpt-4o-mini", scriptedResponses);

var answer = tk.method(SupportAgent::ask)
    .inSession("s-1")
    .invoke("where's order-42");

assertThat(answer).contains("SHIPPED");

// assert that a specific tool was called
assertThat(tk.getToolCalls()).extracting(ToolCall::name)
    .containsExactly("getOrder");
  • Mock model provider - no real API calls, deterministic output
  • Scripted responses drive branching test cases
  • Assert which tools were called, with what args
  • Test system-prompt regressions with fixed model output
  • Real-model tests belong in a separate, opt-in suite
