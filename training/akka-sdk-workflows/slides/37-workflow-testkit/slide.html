WorkflowTestKit: walk the steps.

OrderWorkflowTest.java (excerpt)
var testKit = WorkflowTestKit.of(OrderWorkflow::new);

var started = testKit.method(OrderWorkflow::start)
    .invoke(new StartOrder("c1",
        List.of(new OrderLine("sku-1", 1, 100))));

assertThat(started.getReply()).startsWith("order-");
assertThat(testKit.currentState().totalCents()).isEqualTo(100L);

var transition = testKit.getTransition();
assertThat(transition.stepName()).isEqualTo("reserve");
  • In-JVM. No cluster, no runtime.
  • Feed commands; inspect state after each
  • Assert step transitions and timer schedules
  • Mock component clients via testkit config — or use real ones for integration-style tests
  • Fast: whole saga in milliseconds
