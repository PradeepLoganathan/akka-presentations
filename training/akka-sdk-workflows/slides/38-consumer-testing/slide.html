Testing & ops

Consumers: feed an event, assert a call.

Consumer test shape
var testKit = ConsumerTestKit.of(CustomerAuditConsumer::new);

var result = testKit.onEvent(
    new customerCreated("c-1", "Ada", "Lovelace"));

assertThat(result.getSideEffects()).hasSize(1);
// side effect = the AuditLogEntity.append call
  • Same shape as EventSourcedTestKit: instantiate, feed, assert
  • Side effects captured - you assert on the deferred calls
  • Nothing actually calls the network
  • Test idempotency by feeding the same event twice
  • Combine with a WorkflowTestKit for end-to-end scenarios
