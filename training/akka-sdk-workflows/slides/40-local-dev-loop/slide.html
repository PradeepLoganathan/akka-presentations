The dev loop, day-long edition.

Terminal habits
# start
mvn compile exec:java

# reload on code change: Ctrl+C, re-run
mvn compile exec:java

# run tests without spinning up the runtime
mvn test

# deploy the same code to the managed runtime
akka services deploy customers \
    ghcr.io/pradeepl/akka-workshop:1.0
  • No DB. No Kafka. No Kubernetes to warm up.
  • Console at localhost:9000/akka/console
  • Logs to stdout. Tail with your favourite tool.
  • Same code deploys unchanged to the managed runtime
  • Workflows survive restarts even in dev - the local journal is real
